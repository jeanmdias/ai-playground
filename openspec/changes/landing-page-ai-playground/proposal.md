## Why

O repositório `ai-playground` carece de uma página de entrada que apresente o projeto de forma clara e profissional. Uma landing page em React serve como vitrine do lab, facilitando onboarding e documentação visual do propósito do repositório.

## What Changes

- Criação de uma aplicação React (Vite + TypeScript) dentro de `landing-page/`
- Implementação de uma landing page responsiva com título "ai-playground"
- Seções: Hero, Sobre, Experimentos em destaque e Footer
- Estilização com Tailwind CSS seguindo boas práticas de acessibilidade e responsividade
